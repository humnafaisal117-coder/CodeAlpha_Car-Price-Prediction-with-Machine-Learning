# 🚗 Car Price Prediction using Machine Learning
**📌 Project Overview**

This project builds a Machine Learning regression model to predict the selling price of used cars based on various features such as fuel type, transmission, mileage, and other car-related attributes.

The objective is to understand how machine learning can be applied in real-world price prediction problems.

**🎯 Problem Statement**

Car prices vary depending on multiple factors such as:

* Brand reputation

* Fuel type

* Transmission type

* Mileage driven

* Ownership history

* Present showroom price

**This project aims to predict the selling price of a car using these features.**

* 🛠️ Technologies Used

* 🐍 Python

* 📊 Pandas

* 📈 Matplotlib

* 🤖 Scikit-learn

* 📓 Google Colab

**📂 Dataset**

The dataset contains various car-related features such as:

* Present_Price

* Kms_Driven

* Fuel_Type

* Selling_type

* Transmission

* Owner

* Selling_Price (Target Variable)

**⚙️ Project Workflow**

**1️⃣ Data Loading & Exploration**

* Loaded dataset using Pandas

* Checked dataset structure and column types

* Verified missing values

**2️⃣ Data Preprocessing**

* Encoded categorical variables

* Removed unnecessary columns

* Defined Features (X) and Target (y)

* Split data into training and testing sets

**3️⃣ Model Training**

* Applied Linear Regression model

* Trained model using training dataset

**4️⃣ Model Evaluation**

Model performance was evaluated using:

* Mean Absolute Error (MAE)

* Mean Squared Error (MSE)

* R² Score

The model achieved a high R² score, indicating strong predictive performance.

**5️⃣ Data Visualization**

* 📊 Bar Chart for feature comparison

* 📈 Line Chart for actual vs predicted prices

* 📊 Model Performance

The model produced strong results with high accuracy and good generalization on test data.

This demonstrates how regression models can effectively predict real-world prices.
